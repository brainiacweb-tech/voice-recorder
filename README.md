# Voice Recorder

> Record audio from your microphone and save as WAV files, with instant playback.

---

## Features
- Records microphone input to .wav
- Auto-named with timestamp
- Playback of last recording
- Configurable duration

## Requirements
```
pip install sounddevice numpy soundfile
```

## Installation
```bash
git clone https://github.com/brainiacweb-tech/voice-recorder
cd voice-recorder
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
